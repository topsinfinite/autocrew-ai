AutoCrew Chat Widget Test

Instructions

This page tests the AutoCrew chat widget locally. The widget button should appear in the bottom-right corner.

Note: For the widget to actually send messages, you need a valid n8n webhook URL. The current configuration uses a placeholder URL.

Test Scenarios

  • Click the chat button to open/close the widget
  • Try sending a message (will show error without valid webhook)
  • Check the greeting bubble behavior (refresh with different firstLaunchAction)
  • Test keyboard navigation (Tab, Escape)
  • Resize browser to test mobile responsiveness
